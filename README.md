# My Portfolio Hub

[Live demonstration](https://kenshou2.github.io/andrii-portfolio/)

A personal portfolio website built to showcase my web development projects.  
Designed with **React, TypeScript, and Tailwind CSS**, it features a clean, responsive layout and easy navigation to all my projects.

## Technologies
- React
- TypeScript
- Tailwind CSS

## Features
- Responsive, modern UI for portfolio presentation
- Project cards with links to GitHub repos and live demos
- Smooth navigation between projects
